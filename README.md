-- Student Gradebook CLI --

- Description
A command-line tool to manage student grades, calculate weighted GPA, and persist data using JSON.

- How to Run
+ Option 1:
1. Ensure you have Python installed.
2. Open terminal in the project folder.
3. Run the command: `python main.py`

+ Option 2 (vs code):
1. Install python
2. Install extension call code runner
3. Run the command: click a triangle on top right corner ( run python file in dedicated terminal )

- Features
- Add, Update, Delete courses.
- Auto-save and auto-load data (`gradebook.json`).
- Calculate Overall GPA and Semester GPA.
- Input validation handled.


* CHECK YOUR WINDOW SECURITY (PROTECT HISTORY) IF IT BLOCK THE GRADEBOOK TO UPDATE DATA ON JSON FILE OR NOT -> change it to allow
